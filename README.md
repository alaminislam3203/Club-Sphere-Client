## 🌟 Project Overview

ClubSphere is a modern full-stack platform to manage clubs, memberships, and
events.

- 📸 Photography Clubs
- 🥾 Hiking Groups
- 📚 Book Clubs
- 💻 Tech Communities

---

## ✨ Features

✔ Authentication (JWT)  
✔ Role-based Dashboard (Admin / Manager / Member)  
✔ Stripe Payment Integration  
✔ CRUD Operations  
✔ Secure API

> ⚡ Built with a modern, scalable, and high-performance frontend ecosystem.

## 🎯 Core Technologies

| Category        | Technology             |
| --------------- | ---------------------- |
| ⚛️ Frontend     | React 19               |

| 🎨 Styling      | Tailwind CSS + DaisyUI |
| 🔄 Routing      | React Router v7        |
